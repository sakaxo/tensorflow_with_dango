TensorFlow with Dango

With a training accuracy of 96% and validation accuracy of 68% 60%, this project was my first-ever deep learning (CNN) model built with TensorFlow. Cats, dogs, and parrots were used to train the model, which will eventually be expanded. There are a total of 1800, photos, 600 in each category.

I created this project after being inspired by the recent use of Artificial intelligence and machine learning in almost all over the world.

Project Instructions

  • Install Project Requirements: pip install -r requirements.txt
  • Run python manage.py migrate
  • Run python manage.py collectstatic
  • Run the application: python manage.py runserver
  • And finally, copy and paste this link in your browser http://127.0.0.1:8000/
